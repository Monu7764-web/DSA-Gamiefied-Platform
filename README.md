🎮 DSA Gamified Platform: The Neural Rift
Welcome to DSA Gamified Platform, a high-octane, cinematic coding environment designed to turn Data Structures and Algorithms into an immersive quest. Inspired by the "Solo Leveling" aesthetic, this platform bridges the gap between dry logic and visual mastery.

✨ Features
🌌 1. The Neural Simulations
Don't just code; witness the algorithm in action with our custom-built neon visualizers:

Apex Target Scanner: A high-tech Binary Search simulation with matrix-style scanning effects.

Bubble Sort Neon Flow: Watch elements swap in a glowing, interactive grid.

Recursive Factorial Engine: A "Neural Stack" visualizer that shows how recursion builds and resolves in real-time.

💻 2. Dynamic Dynamic Compiler
In-App Execution: Write Java code directly in the professional-grade editor.

Live Terminal: Real-time feedback with a custom dark-mode console.

Strict Logic Validation: Rewards you only when your logic is 100% efficient.

🛡️ 3. Progression System
Level-Based Quests: Unlock new levels by solving algorithmic challenges.

Solo Leveling Vibe: Cinematic "Level Up" animations and dungeon-style UI elements.

🚀 Getting Started
Prerequisites
JDK 17 or higher (Crucial for the dynamic compiler).

An IDE like IntelliJ IDEA or VS Code.

Installation
Clone the repository:

Bash
git clone https://github.com/Monu7764-web/DSA-Gamiefied-Platform.git
Open the project in your favorite IDE.

Run the Main class:
org.example.Main (or wherever your LoginScreen starts).

🛠️ Project Structure
Plaintext
src/main/java/org/example/
├── engine/        # Problem Database & Logic Engine
├── ui/            # ProblemScreen, LoginScreen & Aesthetic Components
├── simulations/   # BubbleSortSim, TargetSearchSim, FactorialSim
└── Main.java      # Application Entry Point
